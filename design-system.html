Code Usage

So verwendest du das Design System in deinen Projekten.

CSS importieren

<!-- Im <head> deiner HTML-Datei --> <link rel="stylesheet" href="/css/design-system.css">

Fonts einbinden

<!-- Google Fonts --> <link rel="preconnect" href="https://fonts.googleapis.com"> <link href="https://fonts.googleapis.com/css2?family=Outfit:wght@400;500;600;700;800&family=Inter:wght@400;500;600&display=swap" rel="stylesheet"> <!-- Phosphor Icons --> <script src="https://unpkg.com/@phosphor-icons/web@2.0.3"></script>

Button verwenden

<a href="#" class="btn btn-primary"> <i class="ph ph-calendar"></i> Termin buchen </a>

CSS Variables nutzen

.my-component { background: var(--color-surface); border-radius: var(--radius-xl); padding: var(--space-6); box-shadow: var(--shadow-lg); transition: all var(--duration-normal) var(--ease-out); }
